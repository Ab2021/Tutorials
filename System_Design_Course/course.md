# System Design Learning Course

## Goal
Master Large-Scale Distributed System Design in 40 Days.
Focus: Theoretical depth, practical code examples, and real-world case studies.

## Structure
- **Phase 1: Foundations (Days 1-10)**
- **Phase 2: Building Blocks (Days 11-20)**
- **Phase 3: Advanced Architectures (Days 21-30)**
- **Phase 4: Case Studies & Modern Tech (Days 31-40)**

## Syllabus

### Phase 1: Foundations
*   **Day 1:** Scalability Basics (Vertical/Horizontal, Latency/Throughput)
*   **Day 2:** Networking Internals (TCP/UDP, HTTP/1.1/2/3, gRPC)
*   **Day 3:** API Design (REST, GraphQL, RPC, Idempotency)
*   **Day 4:** Database Internals I (ACID, B-Trees, LSM Trees)
*   **Day 5:** Database Internals II (SQL vs NoSQL, CAP, PACELC)
*   **Day 6:** Caching Strategies (Write-through, Write-back, Eviction)
*   **Day 7:** Replication & Sharding (Consistent Hashing, Virtual Nodes)
*   **Day 8:** Distributed Consensus (Paxos, Raft, Zab)
*   **Day 9:** Distributed Transactions (2PC, Sagas, XA)
*   **Day 10:** Review & Foundation Interview Questions

### Phase 2: Building Blocks
*   **Day 11:** Load Balancing (L4 vs L7, Algorithms, HAProxy)
*   **Day 12:** Message Queues (Kafka, RabbitMQ, Pub/Sub patterns)
*   **Day 13:** Key-Value Stores (Redis, DynamoDB internals)
*   **Day 14:** Blob Storage (S3 Architecture, Haystack)
*   **Day 15:** Search Engines (Inverted Index, Elasticsearch, Lucene)
*   **Day 16:** Rate Limiting (Token Bucket, Leaky Bucket, Distributed)
*   **Day 17:** Distributed ID Generation (Snowflake, UUIDs)
*   **Day 18:** Service Discovery (Zookeeper, Consul, Etcd)
*   **Day 19:** Observability (Metrics, Logs, Tracing, Prometheus)
*   **Day 20:** Project: Build a Distributed Rate Limiter (Go/Python)

### Phase 3: Advanced Architectures
*   **Day 21:** Microservices vs Monolith (Patterns, Anti-patterns)
*   **Day 22:** Event-Driven Architecture (Sourcing, CQRS)
*   **Day 23:** Batch Processing (MapReduce, Spark)
*   **Day 24:** Stream Processing (Flink, Storm, Kappa/Lambda)
*   **Day 25:** Content Delivery Networks (CDN, Edge Computing)
*   **Day 26:** Geo-Spatial Indexes (Quadtrees, Geohashes - Uber)
*   **Day 27:** Distributed Locking (Redis Redlock, Chubby)
*   **Day 28:** Security Design (OAuth2, JWT, TLS, DDoS)
*   **Day 29:** Cloud Native Patterns (Kubernetes, Service Mesh)
*   **Day 30:** Project: Build a Mini-KV Store with WAL

### Phase 4: Case Studies & Modern Tech
*   **Day 31:** Design URL Shortener (TinyURL)
*   **Day 32:** Design Pastebin
*   **Day 33:** Design Key-Value Store
*   **Day 34:** Design Chat System (WhatsApp/Discord)
*   **Day 35:** Design Social Feed (Twitter/Facebook)
*   **Day 36:** Design Video Streaming (YouTube/Netflix)
*   **Day 37:** Design Ride Sharing (Uber/Lyft)
*   **Day 38:** Design Search Autocomplete (Typeahead)
*   **Day 39:** Design Vector Database (for AI/LLMs)
*   **Day 40:** Capstone: Design a Global Scale System
