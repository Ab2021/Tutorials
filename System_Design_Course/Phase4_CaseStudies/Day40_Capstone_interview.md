# Day 40 Interview Prep: Final Review

## Q1: What is the CAP Theorem?
**Answer:**
*   **Consistency, Availability, Partition Tolerance.**
*   You can only pick 2.
*   **CP:** Banking (Consistency).
*   **AP:** Social Media (Availability).

## Q2: SQL vs NoSQL?
**Answer:**
*   **SQL:** Structured, ACID, Joins. (Users, Orders).
*   **NoSQL:** Unstructured, Scalable, Flexible. (Logs, Metadata, Feed).

## Q3: How to scale a database?
**Answer:**
*   **Vertical:** Bigger machine.
*   **Read Replicas:** Offload reads.
*   **Sharding:** Split writes.
*   **Caching:** Reduce load.

## Q4: What happens when you type google.com?
**Answer:**
*   DNS -> IP.
*   TCP Handshake.
*   TLS Handshake.
*   HTTP Request.
*   LB -> Web Server -> App Server -> Cache/DB.
*   HTML Response.
*   Browser Rendering.
