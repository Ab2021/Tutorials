# Day 38 Deep Dive: Recent Conference Highlights

## 1. NeurIPS 2023 RL Trends
*   **Foundation Models:** Many papers on LLMs + RL.
*   **Offline RL:** Continued focus on learning from fixed datasets.
*   **Multi-Task Learning:** Generalist agents (GATO, RT-2 style).
*   **Robustness:** Adversarial RL, certified policies.

## 2. ICML 2024 RL Highlights
*   **Transformers:** Scaling up Decision Transformers.
*   **Causal RL:** Using causal inference for better policies.
*   **Sample Efficiency:** Closing the gap with humans and animals.

## 3. ICLR 2024 Observations
*   **Theory:** Improved regret bounds for online RL.
*   **Exploration:** Novel exploration bonuses and algorithms.
*   **Real-World Applications:** More robotics and healthcare papers.

## 4. Notable Recent Papers
*   **Voyager (NVIDIA, 2023):** LLM-powered Minecraft agent that continuously learns.
*   **AlphaDev (DeepMind, 2023):** RL discovers faster sorting algorithms.
*   **Q-Transformer (Google, 2023):** Offline RL for robotics at scale.
