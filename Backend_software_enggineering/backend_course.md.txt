Week 1 — Backend Fundamentals (Days 1–5)

Day 1 – Course Kickoff & Role of Modern Backend Engineer

Topics: Course overview; 2025 backend landscape; distributed, event-driven, AI-powered backends; responsibilities of a backend + AI engineer.

Hands-on: Set up repo, basic folder structure; install language runtime(s) (Python/Node/Go), Docker, Git.

Day 2 – Languages for Backend in 2025

Topics: Python (FastAPI), Node.js, Go, Java/.NET, Rust; pros/cons; where each shines (microservices, AI, infra).

Hands-on: “Hello world” HTTP endpoints in 2 languages (e.g., Python + Node).

Day 3 – HTTP, REST & Web Basics

Topics: HTTP/1.1 vs HTTP/2 vs HTTP/3; request/response lifecycle; headers; methods; status codes; idempotency.

Hands-on: Build a small REST service with 3–4 endpoints, correct methods & status codes.

Day 4 – RESTful API Principles

Topics: Resources vs actions; URL design; statelessness; JSON as default; error modeling.

Hands-on: Design a small REST resource model (Users, Orders); implement CRUD endpoints.

Day 5 – Database Fundamentals Overview

Topics: Relational vs NoSQL vs cache vs time-series vs vector DBs; ACID; CAP; where each fits.

Hands-on: Spin up Postgres + Redis + a vector DB (e.g., Qdrant) using Docker; connect from a small script.

Week 2 — Databases & Auth Foundations (Days 6–10)

Day 6 – Relational Databases & SQL Basics

Topics: Tables, rows, columns; primary keys; basic SELECT/INSERT/UPDATE/DELETE.

Hands-on: Create schema for users & orders; write basic queries.

Day 7 – Normalization, Indexes & Transactions

Topics: 1NF/2NF/3NF, when to denormalize; indexes (single, composite); transactions & isolation levels.

Hands-on: Normalize a simple schema; add indexes; run EXPLAIN ANALYZE on sample queries.

Day 8 – NoSQL: Document & Key-Value Stores

Topics: MongoDB document modeling; embedding vs referencing; Redis structures (strings, hashes, sets).

Hands-on: Model a “user with settings + history” in Mongo; implement simple cache with Redis.

Day 9 – Intro to Vector Databases

Topics: Embeddings, high-dimensional vectors, similarity search; Pinecone/Weaviate/Qdrant overview.

Hands-on: Generate embeddings for sample text with an API or local model; store/query in vector DB.

Day 10 – Auth & Security Basics

Topics: Auth vs authz; JWT; password hashing (bcrypt/Argon2); HTTPS/TLS basics.

Hands-on: Implement user signup/login, hashed passwords, issue/verify JWTs for API endpoints.

Day 15 – Event Sourcing & CQRS (Intro)
