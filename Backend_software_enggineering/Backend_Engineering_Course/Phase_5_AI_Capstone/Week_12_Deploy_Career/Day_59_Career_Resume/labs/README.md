# Lab: Day 59 - Resume Audit

## Goal
Polish your professional profile.

## Step 1: The Template
```text
[Name]
[GitHub] | [LinkedIn] | [Email]

**Skills**
Languages: Python, Go, SQL, JavaScript
Tools: Docker, Kubernetes, AWS, Kafka, Redis, Git

**Experience**
Software Engineer @ [Company]
- Designed and built a microservice handling 50k RPS using FastAPI and Redis.
- Reduced CI/CD build time by 50% by implementing Docker caching.
- Mentored 2 junior engineers.

**Projects**
DocuMind (AI Collaborative Editor)
- Built a real-time document editor with OpenAI RAG and WebSockets.
- Deployed to AWS using Docker Compose and Nginx.
```

## Step 2: The Audit
1.  **Quantify**: Do you have numbers? (%, $, RPS).
2.  **Active Voice**: "Built", "Designed", "Optimized" (Not "Helped with").
3.  **Relevance**: Remove "Microsoft Word" skill. Add "Postgres".

## Step 3: LinkedIn
*   **Headline**: "Backend Engineer | Python, Cloud, AI".
*   **About**: Short summary of your passion for distributed systems.
