## üêû Common Pitfalls
*(Watch out for these common mistakes)*

- **Hardware**: Incorrect wiring, power supply issues, loose connections.
- **Software**: Memory leaks, race conditions, incorrect register configuration, stack overflow.
- **Protocol**: Timing violations, wrong baud rate/clock polarity, missing pull-up resistors.
- **Logic**: Off-by-one errors, infinite loops, unhandled exceptions.
